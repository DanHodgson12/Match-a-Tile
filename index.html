Match-a-Tile

About

The 'Match-a-Tile' game is a memory challenge based on 'Kim's Game' , whereby the player is shown a number of items, which are then covered after a period of time. The player then has to remember the items they were shown and guess what was there. In this version of the game, players must match the tiles at the bottom of the screen with the computer's tiles.

Instructions

Before you begin, select a difficulty mode. The mode will determine the number of turns the player has to match the tiles. For example:

  • Easy: 18 turns
  • Normal: 14 turns
  • Hard: 9 turns

When you are ready, click 'Start' and a 5 second countdown timer will begin. At the start of the countdown timer, the computer tiles will appear - take this time to remember each tile and commit them to memory!

Once the 5 seconds are over, the computer tiles flip over and the player tiles appear. The game will then begin. The player must click on a player tile at the bottom of the screen and match it with the correct computer tile.

The game will end once all tiles have been matched, or when the number of remaining turns reaches zero. Your score will then be displayed in the top right corner of the game area.

If you'd like to reset the game at any point simply click the 'Reset' button. Happy gaming!
